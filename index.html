POT ACCOUNTS

Shared accounting for your team. Everyone sees the same live data. Setup takes 2 minutes.

Step 1 — Get a free JSONBin API key

Go to jsonbin.io, create a free account, then go to API Keys and copy your Master Key.

Step 2 — Bin ID

If you're the first person setting up, leave this blank and we'll create a new bin. If someone else already set it up, paste their Bin ID here.

Your API key is stored only in this browser. Share the Bin ID with teammates so they can connect.
